# php-xdebug-docker-fast 快速构建PHP Debug项目
___
## Step 1
下载项目
```
git clone https://github.com/1oid/php-xdebug-docker-fast
```
将php项目放到此项目下
```
cp -r projects/* php-xdebug-docker-fast
```

## Step 2
```
cd php-xdebug-docker-fast
docker-compose up -d
```

## Step3
配置PHPStorm的Xdebug

## Step4
访问 localhost:8080
